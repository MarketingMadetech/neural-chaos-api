# Git Repository for Neural Chaos Forum

## Initialize and Deploy

```bash
# Initialize git repo
git init
git config user.name "Neural Chaos"
git config user.email "noreply@chaosarchitect.art"

# Add all files
git add .

# First commit
git commit -m "feat: Add static HTML routes for index and admin dashboard"

# Add Render as remote (if using Render native git)
git remote add render <your-render-git-url>

# Or push to GitHub and connect to Render
git remote add origin https://github.com/your-username/neural-chaos-forum.git
git push -u origin main

# Render will auto-detect the push and redeploy
```
